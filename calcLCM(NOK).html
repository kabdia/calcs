Что такое НОК (наименьшее общее кратное)?

Определения из Википедии

1) Наименьшее общее кратное (НОК) двух целых чисел m и n есть наименьшее натуральное число, которое делится на m и n без остатка, то есть кратно им обоим.

2) Наименьшее общее кратное для нескольких чисел — это наименьшее натуральное число, которое делится на каждое из этих чисел.

Определение из foxford.ru

Наименьшим общим кратным двух натуральных чисел называют такое наименьшее натуральное число, которое нацело делится на каждое из данных двух чисел.

Зачем нужно вычислять НОК

НОК помогает:

  1. Приводить дроби к общему знаменателю
  2. Решать задачи на повторение событий (например, когда два автобуса отправляются с интервалами 15 и 20 минут, и нужно узнать, через какое время они снова отправятся одновременно)
  3. Упрощать алгебраические выражения

Как вычислить НОК: основные методы

Рассмотрим основные методы вычисления наименьшего общего кратного. найти наименьшее общее кратное можно как для двух чисел, так и для нескольких чисел.

1) Разложение на простые множители

Инструкция

  • Нужно сначала разложить каждое число на простые множители
  • Затем нужно выписать все простые множители, входящие хотя бы в одно из чисел
  • Каждый из взятых множителей необходимо возвести в наибольшую из тех степеней, с которыми он входит в данные числа
  • Произвести умножение

Рассмотрим на примере:

  • У нас есть три числа - 100, 513, 1200
  • Раскладываем на множители, возводим каждый множитель в степень:
    100 = 2 x 2 x 5 x 5 = 22 x 5 2
    513 = 3 x 3 x 3 x 19 = 33 x 19
    1200 = 2 x 2 x 2 x 3 x 2 x 5 x 5 = 24 x 52 x 3
  • Видим, что среди множителей у нас следующие числа: 2, 3, 5, 19. Выбираем для них самые высокие степени, которые есть в разложениях. У 2 самая большая степень - 4, у 3 - степень 3, у 5 - степень 2, число 19 записываем как есть
  • Считаем 24 x 33 x 5 2 x 19 = 205 200. Наш ответ - 205 200

2) Вычисление НОК через НОД (способ для 2 чисел)

Инструкция

  • Сначала найдем НОД чисел
  • Перемножим исходные числа и разделим результат на НОД

Рассмотрим на примере

  • У нас есть два числа - 100, 1200
  • Найдем НОД. Раскладываем на множители, возводим каждый множитель в степень:
    100 = 2 x 2 x 5 x 5 = 22 x 5 2
    1200 = 2 x 2 x 2 x 3 x 2 x 5 x 5 = 24 x 52 x 3
  • В данном случае у нас общие множители это 2 и 5. У 2 минимальная степень - 2, у 5 - степень 2, 22x52= 100. НОД = 100
  • Перемножим исходные числа и разделим на 1 - (100 x 1200) / 100 = 1200
